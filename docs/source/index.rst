Welcome to conn2res's documentation!
====================================

**conn2res** is a Python library for neuroscientists that allows users to
implement biological connectomes as artificial reservoirs to perform the cognitive
tasks included in the comprehensive corpus of `Neurogym <https://github.com/neurogym/neurogym>_` 
tasks.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::
   
   This project is under active development.
